## 📌 Featured Project

### ♟ Chess Pro V2 — [chess-pro-v2](https://github.com/wali-rehman/chess-pro-v2)

> A professional-grade browser chess engine rebuilt in **Next.js + TypeScript**.

Engineered using Requirements Engineering principles (SENG 321) to ensure 100% adherence to FIDE chess rules, including edge cases like **En Passant**, **Stalemate**, and **Promotion**. The original vanilla JS implementation featured:

- Full move validation with check, checkmate, and stalemate detection
- Undo/redo via saved board states
- Drag-and-drop + click controls with real-time move highlighting
- Move history in standard chess notation
- Clean separation of game state from UI logic (AI opponent-ready architecture)

The V2 refactor migrates all logic into a typed `useChessLogic` hook, adding type-safe `Piece` and `Position` interfaces to catch bugs before they reach runtime.
